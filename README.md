# Slapper Website

Static marketing site for the Slapper game. Built as a simple HTML/CSS landing page with rich media assets (screenshots, backgrounds, character art, social cards).

## Features

- Hero section with game branding and CTA
- Feature highlights and gameplay screenshots
- Responsive layout for desktop and mobile
- Visual assets for social sharing (OG/Twitter cards)
- Game art library (backgrounds, characters, icons)

## Project Structure

- `index.html` - Main landing page markup
- `styles.css` - Site styling
- `assets/` - Images, backgrounds, screenshots, icons, social cards

## Preview

Open `index.html` directly in a browser, or run a simple static server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Asset Notes

- Backgrounds are split into `landscape/` and `portrait/` variants.
- Screenshots and social cards live under `assets/screenshots/` and `assets/social/`.
